# EJEMPLO DE USO - Pixel Check Pro

Para probar el sistema, necesita una imagen dark frame.

## CÓMO OBTENER UN DARK FRAME:

1. Configuración de cámara:
   - Modo Manual (M)
   - ISO más bajo (ej: ISO 100)
   - Tapa puesta en el objetivo
   - Exposición: 1/4000s a 30s

2. Procesamiento:
   - Exportar a TIFF 16-bit SIN compresión
   - Sin ajustes de contraste/brillo
   - Sin reducción de ruido

## FORMATOS SOPORTADOS:
- TIFF 16-bit (RECOMENDADO)
- RAW (CR2, NEF, ARW) - requiere conversión previa
- PNG 16-bit

## RESULTADOS ESPERADOS:

Cámara profesional (ej: Canon 5D Mark IV):
- Hot pixels: 10-100 (0.0001% - 0.001%)
- Dead pixels: 5-50 (0.00005% - 0.0005%)
- Clase ISO: A o A+

Cámara consumer (ej: Canon Rebel):
- Hot pixels: 50-500 (0.0005% - 0.005%)
- Dead pixels: 20-200 (0.0002% - 0.002%)
- Clase ISO: B o C

## NOTA:
Este ZIP NO incluye imágenes reales por tamaño.
Use sus propias dark frames capturadas según instrucciones.
